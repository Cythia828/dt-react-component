# ScrollText 输入框

## 何时使用

数据同步任务字段类型输入框

## 示例

```jsx
import React from 'react';
import { ScrollText } from 'dt-react-component';

export default () => {
    return <ScrollText value="string" />;
};
```

## API

| 参数  | 说明         | 类型     | 默认值 |
| ----- | ------------ | -------- | ------ |
| value | 输入框显示值 | `string` | -      |
